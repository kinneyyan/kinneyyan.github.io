Java7中的NIO.2

2014-12-28

Java7 对原有的 NIO 进行了重大改进，叫 NIO.2。

其中之一就是提供了全面的文件 IO 和文件系统访问支持，新增了 java.nio.file 包及各个子包。

NIO.2 引入了 Path 接口，Path 接口代表一个平台无关的平台路径；还提供了 Files、Paths 两个工具类，Files 包含了大量静态的工具方法来操作文件，Paths 则包含了两个返回 Path 的静态工厂方法。
